## 🔍 Overview
**HotelRevAI** is an advanced data analytics and predictive modeling project developed as part of an internship program at Infosys Springboard. This project leverages historical hotel booking data to uncover meaningful patterns, optimize revenue management, predict cancellations, and provide actionable insights for hotel management teams.

## ✨ Key Features
- **Cancellation Analysis:** Predictive modeling to understand and forecast booking cancellations.
- **Performance Dashboards:** Comprehensive view of overall hotel performance metrics (ADR, RevPAR, Occupancy).
- **DAX Modeling:** Advanced data modeling and custom measures in Power BI for dynamic analysis.
- **Predictive Analytics:** Forecasting future trends to improve resource allocation and pricing strategies.

## 🛠️ Tools & Technologies
- **Microsoft Power BI:** Used for data visualization, dashboard creation, DAX modeling, and predictive reporting. (`DAX Model.pbix`, `predictive analysis.pbix`, `Cancellation analysis.pbix`)
- **Excel / CSV:** For raw data processing and data source management (`hotel_booking.xlsx`, `hotel_booking.csv`).
- **PowerPoint / Word:** Documentation, presentation, and milestone reporting.

## 🚀 Getting Started
To view the models locally:
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/Arunkumar-v43551/infosys-springboard.git
   ```
2. Unzip any `.zip` files (such as `Cancellation analysis.pbix.zip`).
3. Open the `.pbix` files using **Microsoft Power BI Desktop**.
4. Read through the `.docx` and `.pdf` reports for an in-depth understanding of the milestones and methodology.
